一、开机没反应 / 不启动
检查电源：有没有插好、充电座灯亮不亮
长按开机键 10 秒强制重启
机身底部电源开关有没有拨到 ON
放回到充电座重新充电 15 分钟再试

二、能开机但不动 / 走几步就停
边刷、主刷被头发、线缠住 → 清理
轮子卡了异物 → 手动转一下轮子
电池亏电 → 充满再用
防跌落传感器脏了 → 擦一下底部感应器

三、扫不干净 / 吸力变小
尘盒满了 → 倒掉 + 清洗滤网
主刷 / 滚刷毛絮缠绕 → 拆下来清理
风道、吸口堵了 → 疏通
滤网长期没洗 / 老化 → 清洗或更换

四、回不去充电座 / 找不到基站
充电座前方不要有障碍物
充电座靠墙、通电、指示灯正常
擦一下机器人正面导航摄像头 / 感应器
环境太暗也会找不到 → 开灯试试

五、漏扫、乱撞、转圈
激光头 / 摄像头脏了 → 擦干净
地面太光滑 / 反光强 → 挪开镜子、玻璃
建图失败 → 恢复地图 / 重新建图
防撞传感器脏 → 擦拭

六、漏水 / 拖地异常
水箱没卡紧、胶圈歪了
拖布支架没装好
出水孔堵了 → 清水冲洗
不要加清洁剂、消毒液，容易堵泵
通用万能急救步骤
关机 → 清理所有刷子、轮子、传感器
清空尘盒、洗滤网
放回充电座充满电

APP 里恢复出厂 / 清除地图
90% 小故障都能解决。
